Installation
============

Stable release
--------------

To install scrud-django, run this command in your terminal:

```{sourceCode} console
$ pip install scrud-django
```

This is the preferred method to install scrud-django, as it will always install
the most recent stable release.

If you don't have [pip](https://pip.pypa.io) installed, this [Python
installation
guide](http://docs.python-guide.org/en/latest/starting/installation/)
can guide you through the process.

From sources
------------

The sources for scrud-django can be downloaded from the [Github
repo](https://github.com/openteamsinc/scrud-django).

You can either clone the public repository:

```{sourceCode} console
$ git clone git://github.com/openteamsinc/scrud-django
```

Or download the
[tarball](https://github.com/openteamsinc/scrud-django/tarball/main):

```{sourceCode} console
$ curl -OL https://github.com/openteamsinc/scrud-django/tarball/main
```

Once you have a copy of the source, you can install it with:

```{sourceCode} console
$ python setup.py install
```
